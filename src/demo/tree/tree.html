nb-tree

type: Component

Usage

import {TreeModule} from 'xdesign';

Examples

Basic

1.nb-tree用于展现树形数据结构的组件。

2.nb-tree有treeData属性，用于接收treeNode[]值的输入。

3.nb-tree有disabled 属性，用于置灰处理，为不可点选状态。

4.tree-node数据模型TreeNode包含id、name、selectable、isExpanded、isSelected、show、parent、children属性。

其中id: string, name: string, selectable: boolean, isExpanded: boolean, isSelected: boolean, show: boolean, children: TreeNode[],

parent: TreeNodeParent, TreeNodeParent包含parent的id: string。

Tree with modes

1.nb-tree有四种内置selectionMode，分别为：'', 'candidate', 'selected', 'navigation'。

2.nb-tree有propagateSelectionUp和propagateSelectionDown属性，用于设置被点选的node节点是否向上/向下传递选中态。

3.nb-tree有onNodeSelect事件，用于向外抛出被点选节点的数据。
