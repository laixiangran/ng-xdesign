nb-text-line

type: Component

Usage

import {TextLineModule} from 'xdesign';

Examples

Basic

1.nb-text-line用于输入多行文本并自动生成行序号。

2.如果需要特殊样式可以增加class覆盖默认样式。

3.nb-text-line有disabled 属性，用于置灰处理，为不可编辑状态。

4.nb-text-line有value属性，表示当前的值。

TextLine with Form

nb-text-line可以与Forms 和 ReactiveForms 无缝结合使用。
