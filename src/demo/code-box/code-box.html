Basic Buttons

nb-code-box 用于演示某个组件的功能，并显示该功能使用的源代码情况。

标题通过 nb-codebox-title 指令设置，内容描述通过 nb-codebox-desc 指令设置，其余内容为要演示的组件区域。

通过给 nb-code-box 设置 tsCode、htmlCode、lessCode 属性来设置源代码内容。
