nb-table

type: Component

selector: nb-table

exportAs: nbTable

Usage

import {TableModule} from 'xdesign';

Examples

Table Basic

1.nb-table 是一个表格组件。

2.nb-table 核心属性为 datasource，此外还可以提供 theme 属性提供不同表格样式。

3.datasource 为表格的全部数据，另外表格自带分页功能，因此分页后的数据通过 data 属性获取，可以方便的使用模板引用变量来引用。

Table Page

1.表格分页。

2.表格分页可以结合 nb-page 组件来一起使用，表格自生提供了 page 和 pageSize 属性。

3.表格前端分页可以结合 datasource，page 和 pageSize一起使用。

4.表格如果使用后端分页，直接变更 datasource 即可。

Table With Checkbox

可以很方便的在表格中整合 nb-checkbox 组件，实现表格行的选择功能。

Table With Tips, Filters and Sortable

还可以很方便的在表头配置提示、筛选项和排序功能。
配置提示步骤如下：
  • 1. 给表头 th 设置 tipable 属性；
  • 2. 在表头里边设置一个类为 .tip-content 的元素即可。
配置筛选项步骤如下：
  • 1. 给表头 th 设置 filterable 属性；
  • 2. 在表头里边设置一个 [nb-table-filter] 的元素，该元素放置筛选条件；
  • 3. 表头可以添加 filter 事件进行筛选，filterReset 事件进行筛选重置。
配置排序功能如下：
  • 1. 给表头 th 设置 sortable 属性；
  • 2. 在 table 元素上添加 sort 事件；
  • 3. 可以通过 order 和 orderBy 设置表格初始化排序。

Table Column Width Resizable

nb-table 设置 resizable 就可以通过拖动调整表格列宽。

Fix header

此外，还可以实现表头固定的功能。

Fix column

此外，还可以实现表列固定的功能（第一列）。

ColSpan and RowSpan

由于 nb-table 基于原生的 table，因此可以灵活的应用 colspan 和 rowspan 来实现跨行和跨列的处理。

expandable

还可以实现列表的折叠展示功能。

edit

最后，还可以在列表中配置可以编辑的列。
