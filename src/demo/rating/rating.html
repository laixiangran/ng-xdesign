nb-rating

type: Component

selector: nb-rating

exportAs: nbRating

Usage

import {RatingModule} from 'xdesign';

Examples

Rating Basic

nb-rating用于评分，拥有 total、 value、 readonly、 disabled等属性。

Tooltip

可以结合 showTip 开启提示功能，提示的模板由 tipTemplate 属性提供，可以是一个字符串，或者是一个自定义函数。

Themes

可以使用 theme 属性来自定义样式。

Forms

可以和 Forms 和 ReactiveForms 无缝结合使用。
