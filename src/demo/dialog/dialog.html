nb-dialog

type: Component

selector: nb-dialog

exportAs: nbDialog

Usage

import {DialogModule} from 'xdesign';

Examples

Dialog Basic

1.通过 modalable设置对话框为模态对话框，默认为非模态对话框

2.通过 closable设置对话框右上角是否有关闭功能，默认没有

Dialog Content

你可以自定义header、body、footer三部分，其中footer部分如果未自定义，将默认使用【确定】、【取消】两个功能

Dialog Dynamic

动态创建对话框时，必须注意：1.构造函数必须声明viewContainerRef；2.必须在OnDestroy方法中销毁dialogService

动态加载，通过事件绑定的方式 (click)="openSpecialDialog('error', 'you\'ve got an error', 'Error')"动态创建三种对话框，分别是info、error、success
