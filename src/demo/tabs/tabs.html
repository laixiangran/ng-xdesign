nb-tabs

type: Component

selector: nb-tabs

exportAs: nbTabs

Usage

import {TabsModule} from 'xdesign';

Examples

Tabs Basic

1.tabs可以设置 underline ，将在整体底部增加一条下划线作为标签和内容的分隔

2.tabs可以设置 splitline ，标签之间增加竖线作为分隔

3.tabs有两种尺寸 'default' | 'large'，默认情况下是较小尺寸，当显示大尺寸时设置 size="large"

4.tab可以设置 disabled ，用于不可使用状态

Tabs Custom Title

标签支持任意HTML片段、Angular组件、指令，自定义标签写在<ng-template nbTabTitle="XXX"></ng-template>标记内，以文字+问号提示为例说明

特别需要注意的是:

尽管标签是自定义的，仍然需要在tab上配置title="XXX"，值与nbTabTitle的值相同。各个Tab的title是唯一的，不能相同。

使用这个策略的原因是要支持这样的场景：嵌套Tab中含有自定义模版，为了区分自定义的模版属于哪一层级的Tab，需要进行title匹配

Tabs Content

内容支持任意HTML片段、Angular组件、指令，以嵌套tab组件为例说明
