nb-switch

type: Component

selector: nb-switch

exportAs: nbSwitch

Usage

import {SwitchModule} from 'xdesign';

Examples

Switch Basic

1.switch 是 checkbox 的另外一种展现形式，有 checked，disabled 属性，选中情况变更时，会触发 change 事件。

2.switch 可以通过 theme 自定义样式，如果设置 theme="small"，会生成一个 nb-switch-small 的样式类。

Switch With (Reactive)Forms

switch 能够和 Forms 以及 ReactiveForms 无缝使用。
