nb-datepicker

type: Component

selector: nb-month-view, nb-calendar, nb-datepicker, nb-range-datepicker

exportAs: nbMonthView, nbCalendar, nbDatePicker, nbRangeDatePicker

Usage

import {CalendarModule} from 'xdesign';

Examples

MonthView

1.nb-month-view 用于显示月视图。

2.属性 value 为当前选中日期，month 为当前月份所在日期。

3.通过设置属性 multiple 为 true，可以启动多选，同时 startDate 和 endDate 为选中的日期范围。

Calendar

1.nb-calendar 为提供月份切换和年份切换的日历面板。

2.属性 value 为当前选中日期，disabled 可以设置置灰情况。

Date Picker

1.nb-datepicker 为带输入框的单日历选择组件。

2.属性 value 为当前选中日期，disabled 可以设置置灰情况。

Date Range Picker

1.nb-range-datepicker 为带输入框的日历范围选择组件。

2.属性 value 为当前选中日期范围对象，包括 startDate 和 endDate，disabled 可以设置置灰情况。

Date Picker With Forms

nb-datepicker 和 nb-range-datepicker 均可以和 Forms 以及 ReactiveForms 无缝结合使用。
