Usage

import {SpinnerModule} from 'xdesign';

Examples

Spinner Basic

1.nb-spinner 用于输入数字，可以手动输入，也可以通过右侧的增加和渐少按钮输入。

2.nb-spinner 有 value 属性，表示当前的值。

3.nb-spinner 有 step 属性，表示每次增加或渐少的值，默认为1。

4.nb-spinner 有 minValue 和 maxValue 属性，表示输入的限定范围，如果超出限定值，会显示限定侧的最大/最小值。

5.nb-spinner 有 disabled 属性，用于置灰处理。

6.nb-spinner 值变更时，会触发 change 事件，参数为当前值。

Spinner with Forms

nb-spinner 可以和 Forms 和 ReactiveForms 无缝结合使用。
