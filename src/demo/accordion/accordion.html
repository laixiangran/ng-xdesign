nb-accordion

type: Component

selector: nb-accordion

exportAs: nbAccordion

Usage

import {AccordionModule} from 'xdesign';

Examples

Accordion Basic

默认情况下，展开第一项，其他项目处于折叠状态；点击当前项目无反应；点击其他项目时，折叠当前项目，展开点击的项目。

Accordion Collapsible

折叠方式支持互斥折叠和全部可折叠，上一示例为互斥折叠，当前示例为全部可折叠；
互斥折叠：所有项目中只有一个项目可以展开；
全部可折叠：每一项目独自展开折叠，与其他项目的状态无关；
通过设置 collapsible， true 为全部可折叠，false 为互斥折叠。

Accordion Hoverable

支持 click 、hover 两种触发方式，默认情况下是 click，需要 hover 展开时设置 hoverable
